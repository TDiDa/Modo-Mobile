勾选按钮.

## 何时使用

用于标记切换某种状态，如协议勾选.

## API

| 属性         | 说明                      | 类型                 | 默认值  |
| ------------ | ------------------------- | -------------------- | ------- |
| value        | value 值                  | boolean              | 无      |
| defaultValue | 设置初始默认值            | boolean              | -       |
| disabled     | 是否禁用                  | boolean              | `false` |
| onChange     | change 事件触发的回调函数 | (val: boolean): void | -       |
