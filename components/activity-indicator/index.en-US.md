ActivityIndicator.

## When To Use

Activity indicator, generally used for ongoing tasks.

## API

| Property | Description                       | Type    | Default   |
| -------- | --------------------------------- | ------- | --------- |
| type     | can be set to `roller` `carousel` | string  | `Roller`  |
| vertical | vertical                          | boolean | false     |
| color    | color                             | string  | `#1e9eff` |
| size     | size                              | number  | 16        |
