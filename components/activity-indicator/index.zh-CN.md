活动指示器。

## 何时使用

活动指示器，一般用于正在进行中的任务示意。

## API

| 属性     | 说明                      | 类型    | 默认值    |
| -------- | ------------------------- | ------- | --------- |
| type     | 可选值`roller` `carousel` | string  | `Roller`  |
| vertical | 是否垂直显示              | boolean | false     |
| color    | 指示器颜色                | string  | `#1e9eff` |
| size     | 指示器大小                | number  | 16        |
